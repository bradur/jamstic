Slime Janitor

Took out the WebGL build since it was way too buggy.

2018-08-13_00-31-38 Slime Janitor.png

Prevent the slime from taking over the world. Blow it up!

About the builds:

  • Best working build is Windows.
  • Works best in 16:9 resolution from 1280x720 onwards.
  • Linux & Mac builds NOT tested.

Keys

Key(s) Action
Arrows Move
Spacebar Use tool
1, 2, 3 Switch tools
Esc Pause
M Mute music
No mouse controls.
